Submit Your Games

01.24.13 With the help of the GameKnot.com chess viewer, submitting your games to ChessMaine for publication is easier than you may think. In this post, we show you the steps to get your brilliancies - and learning experiences - up on the web. With your help and games we can make ChessMaine, the official site of the Maine Chess Association, even better!

Here's the simple step-by-step instructions to get your games published on ChessMaine:

  1. Click here to go to the GameKnot.com iChess Viewer. Scroll down a bit and click on the orange new game link.
  2. You can either enter all the moves by hand by moving the pieces on the chess board, and enter all the comments (optional) by typing them into the edit box. Alternatively, you can use "Import PGN" link (near the bottom) to load an existing PGN notation for the chess game.
  3. Use the "Edit" link (on the top left of the chessboard) to enter/change the game information, such as the title of the touranment and the player's names and ratings, as desired. Click "OK" to finish editing.
  4. Select the desired move to be displayed initially when the page is loaded by clicking on it in the move list, or by using the "<" and ">" buttons. You can also flip the board using the "Flip" link.
  5. Click on "Web/HTML link (on the bottom of the chessboard) to access the source code. Copy the HTML/JavaScript code by selecting it and then copying it, please make sure to select the entire text (it can be very large if there are many comments). Please do not modify the code in any way, as it won't function properly if changed.
  6. Paste the copied code in an email to info@chessmaine.net for publication.

After you do this a few times, the process will become intuitive. If you have any questions or need assistance, please let us know. Thanks for your help!
